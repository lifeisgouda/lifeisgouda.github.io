  • eval() : 실행 가능한 문자열을 입력 받아 실행한 결과값을 리턴하는 함수이다.

    1>>> eval('1+2')
23
3>>> eval("'hi' + 'a'")
4'hia'
5>>> eval('divmod(4, 3)')
6(1, 1)

    ​

  • append() : 객체를 맨 뒤에 추가해주는 함수이다.

    1>>> x = [1, 2, 3]
2>>> x.append([4, 5, 6])
3>>> print (x)
4[1, 2, 3, [4, 5, 6]]

    ​

  • extend() : 여러개의 값을 확장시킬 수 있는 함수이다. 함수로 전달되는 인수에는 리스트만 올 수 있다.

1>>> x = [1, 2, 3]
2>>> x.extend([4, 5, 6])
3>>> print (x)
4[1, 2, 3, 4, 5, 6]
