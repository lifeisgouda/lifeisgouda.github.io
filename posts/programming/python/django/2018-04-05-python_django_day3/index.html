fizz buzz

  1. 1 ~ 100까지 숫자를 순서대로 출력한다
  2. 그 숫자가 3의 배수일 때는 숫자 대신 fizz
  3. 그 숫자가 5의 배수일 때는 숫자 대신 buzz
  4. 그 숫자가 15의 배수일 때는 숫자 대신 fizzbuzz 를 출력
1for i in range(1, 101):
2    if i % 15 == 0 :
3        print('fizz')
4    elif i % 5 == 0 : 
5        print('buzz')
6    elif i % 3 == 0 :
7        print('fizzbuzz')
8    else : 
9        print(i)
